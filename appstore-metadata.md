## 설명 (Description)

> 4000자 이내. 앱 업데이트 심사 시에만 변경 가능.

2026년 AI 개발의 핵심, AI 에이전트를 LangGraph 중심으로 기초부터 프로덕션까지 체계적으로 학습하는 종합 튜토리얼입니다.

이런 분들을 위해 만들었습니다:
- LLM API 호출 경험이 있고, 다음 단계로 에이전트를 만들고 싶은 개발자
- LangGraph로 프로덕션급 에이전트 워크플로우를 설계하고 싶은 분
- MCP(Model Context Protocol)로 AI와 외부 도구를 연결하고 싶은 분
- RAG, Knowledge Graph를 에이전트에 통합하고 싶은 분
- 멀티 에이전트 시스템을 설계하고 배포까지 하고 싶은 분

20챕터 97섹션으로 구성된 체계적인 커리큘럼:

Part 1 - 에이전트 기초 (Ch1~Ch3)
LLM Tool Calling 메커니즘(OpenAI/Anthropic), ReAct 패턴과 에이전트 루프 직접 구현, 대화 메모리와 상태 관리

Part 2 - LangGraph 핵심 (Ch4~Ch7)
StateGraph 기초, 조건 분기와 동적 라우팅, 체크포인트와 영속적 실행, Human-in-the-Loop 워크플로우

Part 3 - 도구와 프로토콜 (Ch8~Ch11)
커스텀 도구 개발, MCP 서버 구축, MCP 클라이언트와 에이전트 통합, A2A 프로토콜 기초

Part 4 - 에이전트 + RAG (Ch12~Ch14)
Agentic RAG(에이전트가 검색을 도구로 활용), Adaptive RAG와 동적 라우팅, GraphRAG와 Knowledge Graph 연동

Part 5 - 멀티 에이전트 (Ch15~Ch16)
Supervisor/Worker 패턴, CrewAI와 LangGraph 비교

Part 6 - 프로덕션 (Ch17~Ch20)
LangSmith 에이전트 평가, 관찰가능성과 디버깅, 가드레일과 Structured Output, FastAPI 배포와 프로덕션 운영

주요 기능:
- 오프라인 학습 지원 — 네트워크 없이도 모든 콘텐츠 열람
- iCloud 동기화 — 즐겨찾기, 읽기 진행률 디바이스 간 동기화
- 구문 강조 코드 블록 — Python 코드를 한눈에 파악
- Mermaid 다이어그램 — 아키텍처와 흐름을 시각적으로 이해
- 다크 모드 완벽 지원

Forge Series는 개발자를 위한 체계적인 학습 앱 시리즈입니다. 각 주제의 핵심 개념부터 실전 프로젝트까지, 현업에서 바로 활용할 수 있는 실용적 커리큘럼을 제공합니다.
